Apps Link

PDM CHECKSHEET

PDM CHECKSHEET is an application designed for checking machine conditions in predictive maintenance activities within areas OC1, OC2, and FSB. It provides essential tools for conducting thorough inspections, ensuring optimal performance, and supporting predictive maintenance strategies.

AM CHECKSHEET

AM CHECKSHEET is an application designed for efficient finding data input directly connected to the maintenance planner and SAP data. It streamlines the process of capturing and managing asset maintenance information, enhancing coordination between planners and SAP systems.

PRE-PR PROD

The PRE-PR PROD application streamlines the purchasing process by providing tools to manage and edit essential purchasing information. It plays a crucial role in optimizing procurement operations, ensuring efficient stock management, and facilitating seamless purchasing workflows.

Supplies & Part

The Supply & Part Menu is an application used to manage and edit information related to supplies and parts. It plays a crucial role in the maintenance process by providing tools to efficiently handle inventory, order supplies, and manage parts data.

LUBRICATION SISTEM

the application is used to edit the lubrication information used in the maintenance process
